# 303 Frame - Embodied Action

**Inherits from:** `COGNITIVE_EVENT`
**Theoretical Paradigm:** 4E Cognition and Ecological Psychology.

**Definition:** A conceptual frame defining cognition not as internal computation, but as an ongoing, structural coupling between a physical body and an ecological niche.

### Frame Elements (FEs) & Canonical Concepts (Lexical Units)

| Frame Element (Role) | Description | Canonical Concepts Categorized Here |
| :--- | :--- | :--- |
| **`[MORPHOLOGY]`** | The physical capabilities and limits of the agent's body. | Sensorimotor System, Morphological Computation |
| **`[UMWELT]`** | The specific subjective environment accessible to the organism. | Ecological Niche, Task Environment |
| **`[AFFORDANCE]`** | The action possibilities latent in the environment relative to morphology. | [[402 Concept - Affordance\|Affordance]], Graspability, Optic Flow |
| **`[COUPLING]`** | The real-time, dynamic interplay between body and world. | Sensorimotor Contingencies, Structural Coupling, Autopoiesis |
| **`[SCAFFOLD]`** | External structures assimilated to augment the cognitive process. | Epistemic Actions, Cognitive Offloading, Extended Mind |
