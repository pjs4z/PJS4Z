# 401 Concept - Working Memory

> **Definition:** Working Memory is a cognitive system with a limited capacity that is responsible for the temporary holding, processing, and manipulation of information.

## Frame Semantic Context
In our [[300 Frame Semantics Meta-Model of Cognition]], **Working Memory** acts as a Lexical Unit that heavily populates the **[[301 Frame - Information Processing]]**.
- It functionally serves as the **`[STORAGE_BUFFER]`**.
- It holds a **`[REPRESENTATION]`** (e.g., verbal or visuospatial data) so that a **`[PROCESSOR]`** (the central executive) can apply an algorithm to it.

## Theoretical Significance
Working memory is historically rooted in the computational metaphor of the mind (acting exactly like a computer's RAM). Baddeley's highly influential multicomponent model divides it into the Central Executive, Phonological Loop, Visuospatial Sketchpad, and Episodic Buffer. It sits in contrast to Connectionist models, which do not recognize distinct localized buffers.
