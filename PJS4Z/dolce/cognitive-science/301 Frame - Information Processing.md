# 301 Frame - Information Processing

**Inherits from:** `COGNITIVE_EVENT`
**Theoretical Paradigm:** Cognitivism and classical Artificial Intelligence (GOFAI).

**Definition:** A conceptual frame defining cognition as the systematic, rule-based computation and transformation of discrete data into structured internal models.

### Frame Elements (FEs) & Canonical Concepts (Lexical Units)

| Frame Element (Role) | Description | Canonical Concepts Categorized Here |
| :--- | :--- | :--- |
| **`[INPUT_SIGNAL]`** | Raw data gathered from sensory transducers. | Stimulus, Transduction, Bottom-up Afference |
| **`[REPRESENTATION]`**| The formal symbolic data structure standing in for the world. | Schemas, Mental Models, Propositional Logic, Symbol |
| **`[PROCESSOR]`** | The algorithmic mechanism manipulating the representation. | Central Executive, Turing Machine, Syntactic Parser |
| **`[STORAGE_BUFFER]`**| Where representations are maintained across time. | [[401 Concept - Working Memory\|Working Memory]], Episodic Memory, Semantic Networks |
| **`[EXECUTIVE_CONTROL]`**| The allocation of processing resources. | Selective Attention, Inhibitory Control |
| **`[OUTPUT_COMMAND]`**| The executable result of the computation. | Motor Command, Efference Copy |

*(Note: In this frame, "Memory" is a physical place/buffer where "Representations" are stored).*
