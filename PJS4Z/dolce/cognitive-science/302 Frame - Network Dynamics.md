# 302 Frame - Network Dynamics

**Inherits from:** `COGNITIVE_EVENT`
**Theoretical Paradigm:** Connectionism and Artificial Neural Networks.

**Definition:** A conceptual frame defining cognition as the emergent result of continuous activation propagating through a distributed, parallel network of simple units.

### Frame Elements (FEs) & Canonical Concepts (Lexical Units)

| Frame Element (Role) | Description | Canonical Concepts Categorized Here |
| :--- | :--- | :--- |
| **`[NODE]`** | The basic, simple unit of processing. | Artificial Neuron, Perceptron, Hidden Unit |
| **`[CONNECTION]`** | The link determining signal transmission strength. | Synaptic Weight, Edge |
| **`[ACTIVATION_RULE]`**| The mathematical threshold for a node to fire. | Sigmoid Function, ReLU, Action Potential |
| **`[LEARNING_MECHANISM]`**| The rule updating the network's connections over time. | Backpropagation, Hebbian Learning, Synaptic Plasticity |
| **`[GLOBAL_STATE]`** | The emergent pattern of the entire network. | Attractor Basin, Distributed Representation, Graceful Degradation |

*(Note: In this frame, "Memory" does not exist as a storage buffer. It exists as a `[GLOBAL_STATE]` distributed across `[CONNECTIONS]`).*
