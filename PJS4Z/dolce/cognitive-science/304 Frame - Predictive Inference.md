# 304 Frame - Predictive Inference

**Inherits from:** `COGNITIVE_EVENT`
**Theoretical Paradigm:** Predictive Processing and the Bayesian Brain hypothesis.

**Definition:** A conceptual frame defining the mind as a Bayesian statistician, constantly attempting to maintain physiological and epistemic homeostasis by anticipating sensory inputs and minimizing surprise.

### Frame Elements (FEs) & Canonical Concepts (Lexical Units)

| Frame Element (Role) | Description | Canonical Concepts Categorized Here |
| :--- | :--- | :--- |
| **`[GENERATIVE_MODEL]`**| The internal top-down guess of the hidden causes of the world. | Prior Probability, Top-Down Prediction, Internal Model |
| **`[SENSORY_EVIDENCE]`**| The actual bottom-up state of the world bounding the model. | Likelihood, Sensory Perturbation, Markov Blanket |
| **`[ERROR_SIGNAL]`** | The discrepancy between the model and the evidence. | [[403 Concept - Prediction Error\|Prediction Error]], Surprise, Variational Free Energy |
| **`[RESOLUTION_POLICY]`**| The mechanism of minimizing the error. | Active Inference, Perceptual Inference |
| **`[MODULATOR]`** | The mechanism determining the reliability of the error signal. | Precision Weighting, Dopaminergic Gain, Attention |

*(Note: In this frame, "Attention" is completely redefined. It is not an `[EXECUTIVE_CONTROL]` spotlight, but rather a `[MODULATOR]` that determines how much mathematical weight to give an `[ERROR_SIGNAL]`).*
